# MinimalistWallpaperApp
Play Store'da yayınlamak için yapılmış olan minimalist resimler barındıran bir Wallpaper App. Firebase kullanılmıştır. Favorilere ekleme, karanlık mod gibi özellikler eklenmiştir.
